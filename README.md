## 프로젝트 설명
>라이브커머스를 통해 많은 사람들이 판매할 물건들을 홍보한다. 이러한 라이브커머스를 반대로 온라인 쇼핑몰 내부에 적용해서 사람들이 온라인 쇼핑몰에서 물건을 쇼핑 할 때 해당 물건에 대한 정보를 좀 더 쉽게 알기 위해서 기록되어 있던 라이브커머스를 보거나 또는 진행 중인 라이브 커머스를 보도록하면 쇼핑몰의 홍보를 도와주기 위한 웹사이트 제작 프로젝트
